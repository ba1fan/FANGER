FANGER（乐享）
======

项目简介：
---
FANGER（乐享）是一款基于信息分享的微型生活感悟平台。
帮助用户“乐分享，爱生活”是本应用的一个宗旨。

关于本项目技术：
---
    1.Spring MVC
    2.JPA规范（hibernate 实现）
    3.MySql数据库
    4.使用maven自动构建
    5.Junit测试（4.0）
    6.Twitter推出的bootStrap工具包
    7.GMap3。
